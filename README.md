# Tokimk


## Installation
```
brew install --HEAD fuyutarow/tokimk/tokimk
```

clean uninstall
```
brew uninstall fuyutarow/tokimk/tokimk
brew untap fuyutarow/tokimk
```


## Usage

- Build from ~/.tokimk.toml
  ```
  tokimk
  ```
- Build from sample.toml
  ```
  tokimk sample.toml
  ```


## Development
```
cargo make hot
```
